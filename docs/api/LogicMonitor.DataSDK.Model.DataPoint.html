AggregationType

The aggregation method, if any, that should be used if data is pushed in sub-minute intervals. Allowed options are “sum”, “average” and “none”(default) where “none” would take last value for that minute. Only considered when creating a new datapoint.See the About the Push Metrics REST API section of this guide for more information on datapoint value aggregation intervals.
